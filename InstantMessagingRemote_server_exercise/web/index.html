Instant Messaging Remote Server

The Instant Messaging backend server application is running.

This server provides RESTful and WebSocket services for the messaging clients.

Available Services:

  • REST API Base Path: /InstantMessagingRemote_server_exercise/rest
    (Provides endpoints for Topic Management and Publishing under /topicmanager and /publisher)
  • WebSocket Endpoint: /InstantMessagingRemote_server_exercise/ws
    (Used for establishing connections for subscribing and receiving messages)
